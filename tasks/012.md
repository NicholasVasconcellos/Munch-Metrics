# Task 012: Interactive Tutorial

## Context

This task builds the multi-step interactive tutorial modal with animated React component demos. It's triggered from the landing page "See How It Works" CTA and a persistent floating help button. Depends on Task 011 (landing page) since the tutorial is launched from the hero CTA. The tutorial demos are live animated React components, not videos or GIFs.

## Steps

1. Create `src/components/tutorial/tutorial-modal.tsx`:
   - Multi-step modal using shadcn/ui Dialog
   - Previous/Next navigation buttons and step dot indicators
   - "Got it" / "Skip" button to dismiss at any time
   - Dynamically imports tutorial steps (code splitting)
2. Create `src/components/tutorial/tutorial-step.tsx`:
   - Template component for a tutorial step: instruction text (1-2 sentences) + animated demo component
   - Accepts title, description, and animation component as props
3. Create 6 animated tutorial demos (live React components with CSS/Framer Motion animations):
   - `search-anim.tsx` — miniature search bar with auto-typing "chicken", autocomplete dropdown appearing
   - `filter-anim.tsx` — filter panel slides open, checkboxes toggle on, filter chips appear
   - `sort-anim.tsx` — column header clicks, arrow indicator animates, rows visually reorder
   - `group-anim.tsx` — group-by dropdown selects "Food Group", rows animate into collapsible groups
   - `columns-anim.tsx` — column picker opens, columns toggle on/off, table width adjusts
   - `export-anim.tsx` — CSV download button pulses, URL copies to clipboard with toast notification
4. Create `src/components/tutorial/tutorial-trigger.tsx`:
   - Floating help button (bottom-right corner, persistent)
   - First-time visitors see a subtle prompt to open the tutorial (dismissable)
   - Tutorial dismissed state stored in localStorage
5. Wire the hero "See How It Works" CTA to open the tutorial modal
6. Write test script to `tests/012_tutorial.sh`

## Test Suggestions

- Tutorial modal opens from hero CTA and floating help button
- All 6 steps are navigable via Previous/Next
- Step dot indicators reflect current step
- Each animation renders and plays (no runtime errors)
- "Skip" closes the modal at any step
- First-visit prompt appears and is dismissable
- Dismissed state persists in localStorage
- Tutorial modal is code-split (not in main bundle)
- TypeScript compiles with no errors

## Definition of Done

The interactive tutorial modal has 6 steps, each with instruction text and a live animated React demo showing the described feature. Navigation (Previous/Next/Skip/step dots) works. A floating help button provides persistent access. First-time visitors see a prompt. Dismissed state persists in localStorage. All animations are smooth and clearly illustrate their feature.
