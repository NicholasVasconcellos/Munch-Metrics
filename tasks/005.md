# Task 005: Tag Derivation Seed Script

## Context

This task creates the script that derives dietary, allergen, and processing tags from the seeded food and nutrient data. It populates the `food_tags` table. Task 003 (USDA seed) must be complete since tags are derived from food_group and nutrient data. This task can run in parallel with Task 004 (prices).

## Files to Read

- PROJECT.md
- src/lib/db/index.ts
- src/lib/db/schema.ts
- src/lib/constants.ts (if it exists from Task 001 — otherwise create tag-related constants here)

## Files to Write

- scripts/seed-tags.ts
- tests/005_seed_tags.sh

## Steps

1. Create `scripts/seed-tags.ts` that connects to the database and reads all foods with their nutrient profiles and food_groups
2. Implement dietary tag heuristics:
   - **Vegan/Vegetarian:** Map by food_group (exclude meat, poultry, fish, dairy, eggs food groups for vegan; exclude only meat/poultry/fish for vegetarian)
   - **Pescatarian:** Exclude meat and poultry food groups but allow fish
   - **Keto-friendly:** Tag foods where carbs_per_100g < 10 and fat_per_100g > 10
   - **High-protein:** Protein_per_100g > 20
   - **Low-sodium:** Sodium < 140mg per 100g
   - **Gluten-free:** Exclude grain/wheat-based food groups; note this is best-effort
   - **Dairy-free:** Exclude dairy food group
3. Implement allergen tag heuristics:
   - Map food groups and descriptions to common allergens (contains_dairy, contains_gluten, contains_nuts, contains_soy, contains_eggs, contains_fish, contains_shellfish)
4. Implement processing classification:
   - Classify based on food_group and description keywords (whole_food, minimally_processed, processed)
5. Bulk insert all derived tags into `food_tags` table
6. Use ON CONFLICT or clear-and-reinsert strategy for idempotency
7. Log summary statistics (count of tags by type and value)
8. Make runnable via `pnpm tsx scripts/seed-tags.ts`
9. Write test script to `tests/005_seed_tags.sh`

## Test Suggestions

- Chicken breast is tagged as: high-protein, dairy-free, gluten-free, contains-none of soy/nuts
- Tofu is tagged as: vegan, vegetarian, contains_soy
- White rice is tagged as: vegan, vegetarian, gluten-free, dairy-free
- Salmon is tagged as: pescatarian, high-protein, contains_fish
- Tag counts are reasonable (e.g., vegan count < total food count)
- Re-running doesn't create duplicate tags

## Definition of Done

Running `pnpm tsx scripts/seed-tags.ts` derives dietary, allergen, and processing tags for all seeded foods using heuristics based on food_group and nutrient data. Tags are inserted into the `food_tags` table. A summary of tag distribution is logged.
