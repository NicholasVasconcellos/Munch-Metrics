# Task 009: Grouping, CSV Export & URL Sharing

## Context

This task adds group-by functionality, CSV export, and the copy-link sharing feature. These are table toolbar features that complete the core table experience. Depends on Task 007 (core table). Can run in parallel with Tasks 008 and 010.

- tests/009_grouping_export.sh

## Steps

1. Create `src/components/table/group-selector.tsx`:
   - Dropdown selector in the table toolbar: "Group by: [None | Food Group | any column]"
   - When active, the searchFoods query adds ORDER BY the group column
   - Render group headers as collapsible rows with group name and count
   - Groups are expandable/collapsible
   - Group state serialized to URL (?group=food_group)
2. Create `src/lib/csv-export.ts`:
   - Transform current table data (filtered/sorted view) to CSV format
   - Respect visible columns only
   - Handle special characters, commas, quotes in values
   - Return as downloadable Blob
3. Create `src/components/table/export-csv.tsx`:
   - Button in table toolbar
   - Exports current view as CSV
   - Filename: `munch-metrics-export-YYYY-MM-DD.csv`
   - Downloads via browser download API
4. Add a "Copy Link" button to the table toolbar:
   - Copies the current URL (with all state params) to clipboard
   - Shows a toast notification on success
5. Verify full URL state round-trip — loading a shared URL reproduces the exact table state (query, sort, direction, columns, filters, group, page)
6. Write test script to `tests/009_grouping_export.sh`

## Test Suggestions

- Group-by selector shows all available columns
- Selecting "Food Group" groups the table rows with collapsible headers
- Group headers show count of items per group
- Collapsing/expanding groups works
- Group state persists in URL (?group=food_group)
- CSV export includes only visible columns
- CSV correctly escapes special characters
- CSV filename includes current date
- Copy Link copies the full URL with state params
- Loading a shared URL restores exact table state

## Definition of Done

Group-by functionality works with collapsible group headers showing counts. CSV export downloads the current filtered/sorted/column-selected view. Copy Link copies the full stateful URL. All state round-trips through URL params — a shared link reproduces the exact same view.
