# Task 010: Mobile Layout & Food Detail Modal

## Context

This task creates the mobile-responsive table layout and the food detail modal. The mobile layout replaces the table with collapsible summary cards on small screens. The food detail modal shows a complete nutrient breakdown when clicking a food name. Depends on Task 007 (core table). Can run in parallel with Tasks 008 and 009.

## Steps

1. Create `src/hooks/use-media-query.ts` — hook that returns true/false for a media query (e.g., `(max-width: 768px)`)
2. Create `src/components/table/mobile-row.tsx`:
   - Replaces table rows when screen width < 768px
   - Each food shows as a summary card: small thumbnail, Name, Food Group, Calories, Protein
   - Tapping a row expands it to show all visible columns as a key-value list
   - Expand/collapse with smooth animation
3. Update `src/components/table/data-table.tsx` to conditionally render mobile-row layout vs. desktop table based on the media query hook
4. Create `src/components/food/food-image.tsx`:
   - Displays food image from the image proxy route
   - Shows blur placeholder while loading (Next.js Image component)
   - Falls back to category-level placeholder icon on error
   - Includes Unsplash photographer attribution link below image
5. Create `src/components/food/food-detail-modal.tsx`:
   - Triggered by clicking a food name in the table
   - Calls `getFoodDetail` server action to fetch full data
   - Displays: large image (food-image component), full name, description, serving size
   - Complete nutrient breakdown organized by category: Macros → Vitamins → Minerals → Amino Acids → Fatty Acids
   - Each nutrient shows amount, unit, and % Daily Value where applicable
   - Price estimate and source label ("BLS actual" vs "category estimate")
   - Computed metrics: calories/$, protein/$
   - Uses shadcn/ui Dialog component
6. Wire the food name column in the table to open the detail modal on click
7. Write test script to `tests/010_mobile_detail.sh`

## Test Suggestions

- Mobile layout activates at viewport < 768px
- Mobile cards show thumbnail, name, food group, calories, protein
- Tapping a mobile card expands to show all visible columns
- Food detail modal opens when clicking a food name
- Modal shows full nutrient breakdown organized by category
- Modal displays price estimate with source label
- Modal shows food image with Unsplash attribution
- Modal closes on backdrop click or X button
- Food image shows placeholder on error/missing image
- TypeScript compiles with no errors

## Definition of Done

On mobile viewports, the table switches to collapsible summary cards that expand to show full details. The food detail modal shows complete nutrient profiles organized by category, price data with source labels, computed metrics, and attributed food images. Both components integrate with existing table infrastructure.
